About JewelSort

JewelSort is a soothing yet strategic color‑sorting puzzle. Pour jewels between cases to group identical colors, minimize moves, and beat the clock.

Key Features

  • Clean, tactile sorting gameplay
  • Smart hints and limited undo (unlimited with Pro)
  • Multiple difficulties: Normal, Hard, Expert
  • Offline play, no accounts, no ads

Quick Tips

  • Use empty cases to stage colors
  • Stack same colors to maximize pours
  • One move undo saves mis‑taps

Need help? Contact nekomura@liz-aria.com.

JewelSort について

JewelSort は、落ち着いた手触りと戦略性を両立した色分けパズル。ケース間で宝石を注ぎ分け、同色をまとめ、最小手数やタイム更新を狙いましょう。

主な特徴

  • シンプルで気持ちよい整理ゲームプレイ
  • 賢いヒントとアンドゥ（プロは無制限）
  • ノーマル/ハード/エキスパートの難易度
  • オフライン対応、アカウント不要、広告なし

攻略のコツ

  • 空ケースを“待避所”として活用
  • 同色を重ね、注げる量を最大化
  • 1手アンドゥで誤タップをリカバー
